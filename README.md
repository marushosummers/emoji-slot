# [emoji-slot](https://emoji-slot.marusho.io/)

====

emoji slot is made for learning React.js/Next.js.

## Requirement

- React.js
- Next.js

## Features

- Generate dynamic OGP with slot results on Twitter.
## Usage

- 開発環境の起動

```
make
```

- 開発環境に入る

```
make enter
cd emoji-slot
yarn dev
```


- 開発環境を止める


```
make stop
```

## Deploy

To deploy on Vercel, push to main branch.
